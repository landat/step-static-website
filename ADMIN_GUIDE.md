# Инструкция администратора

Эта инструкция для человека, который не работает с кодом и хочет с другого компьютера добавить новое видео Rutube в каталог сайта.

## Как добавить видео

1. Войти в GitHub.
2. Открыть репозиторий `landat/step-static-website`.
3. Открыть файл `content/videos.js`.
4. Нажать кнопку редактирования с иконкой карандаша.
5. Перед строкой `];` вставить блок:

```js
{
  title: "Название видео",
  rutube: "https://rutube.ru/video/ID_РОЛИКА/",
  description: "Короткое описание видео"
},
```

6. Нажать `Commit changes`.
7. Подождать 2-5 минут.
8. Проверить страницу `https://landat.github.io/step-static-website/videos.html`.

## Важно

- Кавычки вокруг текста удалять нельзя.
- Запятая после блока нужна, если дальше будет еще одно видео.
- Можно вставлять обычную ссылку Rutube из адресной строки браузера.
- Если сайт не обновился сразу, нажмите `Ctrl+F5`.

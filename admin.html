Как добавить видео Rutube

Инструкция рассчитана на человека, который работает с другого компьютера и не пользуется терминалом.

Добавление видео через GitHub

1. Войдите в аккаунт GitHub, у которого есть доступ к репозиторию сайта.

2. Откройте репозиторий landat/step-static-website.

3. Откройте файл content/videos.js и нажмите кнопку редактирования с иконкой карандаша.

4. Вставьте новый блок перед строкой ];:

{
  title: "Название видео",
  rutube: "https://rutube.ru/video/ID_РОЛИКА/",
  description: "Короткое описание видео"
},

5. Нажмите Commit changes. Через несколько минут GitHub Pages автоматически передеплоит сайт.

6. Проверьте страницу Видео.

Где взять ссылку Rutube

Откройте ролик на Rutube и скопируйте адрес из строки браузера. Подойдет обычная ссылка вида https://rutube.ru/video/....

Если сайт не обновился сразу

Подождите 2-5 минут и обновите страницу с очисткой кеша: Ctrl+F5. GitHub Pages иногда публикует изменения не мгновенно.
